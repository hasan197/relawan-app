
  # Volunteer Dashboard Design

A modern, responsive volunteer management dashboard built with React, TypeScript, and Tailwind CSS. This project provides an intuitive interface for managing volunteers, events, and activities.

## üöÄ Getting Started

### Prerequisites

- Node.js 16.14.0 or later
- npm 8.0.0 or later

### Installation

1. Clone the repository
   ```bash
   git clone [repository-url]
   cd volunteer-dashboard
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser
